Detailed Description

Třída pro tvorbu a správu interaktivních grafických tlačítek.

  • Umožňuje snadné vykreslování tlačítek s textem, změnu barev a detekci dotyku v definované oblasti. Ideální pro ovládací panely a menu.
  • Button btn(10, 10, 100, 50, "Start", tft);
    btn.draw();
    if (btn.isTouched()) {
    // Akce při stisku
    }
