Detailed Description

Abstraktní rozhraní (Interface) pro moduly aplikací.

  • Každá nová aplikace (např. Meteostanice, Přehrávač, Kalkulačka) musí dědit z této třídy a implementovat její virtuální metody. To umožňuje MainScreenu pracovat s různými aplikacemi jednotným způsobem.
