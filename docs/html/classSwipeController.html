Detailed Description

Ovladač pro detekci dotykových gest (přejetí prstem) na displeji.

  • Sleduje trajektorii dotyku od stisknutí po uvolnění a vyhodnocuje směr pohybu (nahoru, dolů, doleva, doprava).
