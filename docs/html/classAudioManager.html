Detailed Description

Třída pro správu přehrávání audia (MP3/WAV) z SD karty přes I2S rozhraní.

  • Instance vyžaduje existující objekt FileLoader. Příklad inicializace:
    FileLoader loader;
    AudioManager audio(loader);
    * void setup() {
    loader.begin();
    audio.begin();
    audio.playMusic("/track.mp3");
    }
