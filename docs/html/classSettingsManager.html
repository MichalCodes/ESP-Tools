Detailed Description

Správce obrazovky nastavení a uživatelského rozhraní pro audio.

  • Tato třída vykresluje ovládací prvky (tlačítka, posuvníky) a propojuje uživatelský vstup přímo s instancí AudioManager.
  • SettingsManager settings(audioManager);
    // V hlavní smyčce nebo při stisku tlačítka "Nastavení":
    settings.show();
    while(settings.isActive()) {
    settings.update();
    }
