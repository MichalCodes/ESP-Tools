Detailed Description

Správce hlavní obrazovky (Launcheru), který vykresluje ikony aplikací a menu.

  • Třída dynamicky spravuje registrované moduly a umožňuje jejich spouštění. Příklad inicializace:
    MainScreen home(tft, "Moje ESP32", settings);
    home.addModule(weatherApp);
    home.draw();
