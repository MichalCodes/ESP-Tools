Detailed Description

Třída pro simulaci analogového joysticku na dotykovém displeji.

  • Tento prvek umožňuje ovládat hry nebo aplikace pomocí virtuálního joysticku. Zajišťuje vykreslení základny (outer) a pohyblivého středu (inner), včetně logiky pro omezení pohybu v kruhu a automatického centrování po uvolnění.
