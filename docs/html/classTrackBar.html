Detailed Description

Grafický ovládací prvek (posuvník) pro výběr číselné hodnoty v daném rozsahu.

  • Umožňuje uživateli měnit hodnoty tažením nebo kliknutím na lištu.
  • TrackBar volumeBar(tft, 50, 100, 200, 40, 0, 100, 50);
    volumeBar.draw();
    * // V obsluze dotyku:
    if (volumeBar.handleTouch(tx, ty)) {
    int newVol = volumeBar.getValue();
